A digital yard sale, built for Claude.

Drop in your photos, it writes the listings. Share the link. Buyers reserve what they want. Host it free on your own GitHub, or let us host it for pennies. Hand-editing works too. All open source.

See a live example.

Two ways to host it.

Host it yourself.

Fork the template and push to GitHub Pages. Free. Same data shape as the hosted version, so you can move between them.

Sign up, add items, share the link. Free during the invite-only beta; about $0.10 a month after. No commission on any sale.

Two ways to build it.

With Claude.

Connect over MCP from Claude Desktop and ask it what you'd ask a friend:

Here are photos of stuff I'm selling. Add them to my sale and suggest prices.
Added 2 items.
Vintage road bike, 56cm
$320

Published as @yrdsl/mcp on npm. Works with either hosting option.

By hand.

Edit two JSON files: site.json for the sale, items.json for the stuff. Clean data shape, no lock-in. 

{
  "title": "Vintage road bike, 56cm",
  "price": 320,
  "image": "photos/bike.jpg",
  "tags": ["bikes", "outdoor"]
}
