# @yrdsl/themes

Plain-CSS themes for [`@yrdsl/viewer`](https://www.npmjs.com/package/@yrdsl/viewer):
`conservative`, `artsy`, `hip`, `retro`.

## Install

```bash
npm install @yrdsl/themes
```

## Usage

Import the stylesheet for the theme you want alongside the viewer's base
styles:

```ts
import '@yrdsl/viewer/styles.css';
import '@yrdsl/themes/conservative';
```

A manifest at `@yrdsl/themes` lists all four themes with ids and display
names.
